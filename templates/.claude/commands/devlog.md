# /devlog - 개발 로그 작성

오늘 날짜로 DEVLOG.md에 새 엔트리를 추가해줘.

## 수행할 작업

1. **오늘 커밋 확인**
   - `git log --oneline --since="00:00"` 로 오늘 커밋 목록 확인
   - 커밋 메시지에서 주요 변경사항 추출

2. **DEVLOG.md 업데이트**
   - 파일 상단에 오늘 날짜 섹션 추가 (기존 내용 위에)
   - 포맷:
   ```markdown
   ## YYYY-MM-DD
   ### 작업 내용
   - [커밋 내용 기반 작업 목록]

   ### 학습/메모
   - [대화 중 학습한 내용이나 메모]

   ### 이슈/해결
   - [있다면 기록]

   ---
   ```

3. **중복 체크**
   - 오늘 날짜 섹션이 이미 있으면 내용만 업데이트
   - 새로운 커밋이 있으면 작업 내용에 추가

## 참고
- 커밋이 없어도 대화 내용 기반으로 작업 내용 작성 가능
- 사용자에게 추가할 학습/메모 내용이 있는지 물어볼 것
