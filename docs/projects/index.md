## 학습 기록 작성 방식

각 프로젝트 문서는 다음 구조로 작성됩니다:

1. **프로젝트 개요** - 목적, 기술 스택
2. **주요 기능 구현** - 기능별 학습 내용
3. **핵심 개념 정리** - 새로 배운 개념들
4. **코드 예시** - 실제 구현 코드
5. **회고** - 느낀 점, 개선할 점
